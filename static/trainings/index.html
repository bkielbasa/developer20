Go - fundaments

×

The training covers all required topics to help your team quickly start using Go on production starting from uderstanding Go syntax to writing first production-ready application.

The course contains:

  • Variables and constants
  • Data structures (arrays, slices, maps, structures)
  • Interfaces and working with them
  • Goroutines and concurrency
  • Channels
  • Data races
  • Testing and benchmarking
  • Context
  • Working with HTTP
  • testing

The course takes two-three days. During this time, you'll learn basics of the language and have practical excercises that are inspired by production code.

Go - advanced

×

Do you know basics and want to know more about the language? In this course, you'll learn more advanced aspects of the language including:

  • understanding GC
  • profiling and tracing
  • optimizing application (micro and macro benchmarks)
  • different synchronization patterns (advanced concurrency)
  • Go good practices

The course takes two-three days. During this time, you'll learn how to write faster and more reliable code.

Microservices in Go

×

Do you know basics and want to know how to write microservices in Go? In this course, you'll learn:

  • gRPC (basics, unary RPC, client and server streaming, bidirectional streaming)
  • communication using message bus
  • add observability to your application (opentelemetry)
  • working with Docker
  • tips about structuring the code
  • optimizing application (micro and macro benchmarks)

The course takes two-three days. During this time, you'll learn how to write reliable and well managed Go applications.
